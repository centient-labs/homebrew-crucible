# Homebrew Tap for Crucible

Homebrew formulae for installing [Crucible](https://github.com/centient-labs/crucible), a pipeline execution engine for Claude Code (MCP server).

## Install

```bash
brew install centient-labs/crucible/crucible
```

Or tap first, then install:

```bash
brew tap centient-labs/crucible
brew install crucible
```

## Upgrade

```bash
brew upgrade crucible
```

## What is Crucible?

Crucible is an MCP server that adds structured pipeline execution capabilities to Claude Code. It provides commands for multi-stage workflows: conceptualize, architect, design, plan, implement, document, and deploy.

For more information, see the [main repository](https://github.com/centient-labs/crucible).
